# 경쟁사 분석 — 공매도 데이터 분석 도구

## 1. 공매도닷컴 (shortselling.kr)

| 항목 | 내용 |
|------|------|
| **서비스 유형** | 웹 기반 공매도 데이터 조회 |
| **가격** | 무료 |
| **데이터** | KRX 공매도 잔고, 거래대금, 대차잔고 |
| **강점** | 무료, 직관적 UI, 당일 데이터 반영 빠름, 종목별 추이 차트 |
| **약점** | 시그널 탐지 없음, 알림 기능 없음, 백테스팅 불가, API 미제공 |
| **차별화 기회** | 자동 시그널 탐지 + 알림 + 백테스팅 |

## 2. DeepSearch (deep-search.io)

| 항목 | 내용 |
|------|------|
| **서비스 유형** | AI 기반 투자 분석 플랫폼 |
| **가격** | 유료 (월 $50+) |
| **데이터** | 재무, 뉴스, 공시, 공매도 포함 다양한 데이터 |
| **강점** | AI 인사이트, 자연어 질의, 뉴스 감성 분석, 종합 분석 |
| **약점** | 비쌈, 공매도 특화 아님, 대차잔고 시그널 없음, 백테스팅 제한적 |
| **차별화 기회** | 대차잔고/숏스퀴즈 전문 특화 + 무료(자가운영) |

## 3. 퀀트킹 (quantking.co.kr)

| 항목 | 내용 |
|------|------|
| **서비스 유형** | 퀀트 투자 백테스팅 플랫폼 |
| **가격** | 부분 유료 |
| **데이터** | 재무 팩터, 기술적 지표, 시장 데이터 |
| **강점** | 다양한 팩터 백테스팅, 포트폴리오 시뮬레이션 |
| **약점** | 공매도/대차잔고 특화 아님, 시그널 알림 없음, 커스텀 데이터 불가 |
| **차별화 기회** | 대차잔고 전용 백테스팅 + 실시간 시그널 |

## 5. 결론

기존 서비스들은 공매도 데이터를 "보여주는" 데 그치거나, 범용 분석 도구에 공매도를 부분 포함하는 수준입니다.
**대차잔고 특화 시그널 탐지 + 백테스팅 + 자동 알림**을 통합한 도구는 현재 시장에 없습니다.
