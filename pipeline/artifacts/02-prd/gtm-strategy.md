# GTM 전략 — 공매도 커버링 시그널 탐지 시스템

> **참고**: 본 시스템은 개인 내부 도구로 외부 마케팅/판매 계획이 없습니다.
> 이 GTM 전략은 향후 v2~v3에서 SaaS 전환을 고려할 때의 참고 자료입니다.

## 1. 포지셔닝

**카테고리**: 공매도 데이터 분석 도구
**차별화**: 대차잔고 특화 시그널 + 자동 알림 + 백테스팅 통합

### 포지셔닝 매트릭스

```
                    시그널 자동 탐지
                         ▲
                         │
       퀀트킹 ●          │         ● 본 시스템
                         │
───────────────────────────────────── 공매도 특화
                         │
       DeepSearch ●      │    ● 공매도닷컴
                         │
                    수동 분석
```

## 2. AARRR 퍼널 (SaaS 전환 시)

| 단계 | 전략 | KPI |
|------|------|-----|
| Acquisition | SEO ("공매도 분석", "대차잔고 조회"), 투자 커뮤니티 (에펨코리아, 뽐뿌 주식) | MAU |
| Activation | 무료 체험 (7일), 첫 시그널 수신까지 < 3분 | 가입→시그널 수신 전환율 |
| Retention | 일일 텔레그램 알림, 주간 백테스팅 리포트 | DAU/MAU ratio |
| Revenue | 프리미엄 플랜 (실시간 알림, 고급 스코어링) | MRR |
| Referral | 시그널 적중 사례 공유 기능 | K-factor |
