| 2. 탐색 | 웹 대시보드 | 전체 시그널 리스트 | 관심 종목 필터링 |

| 4. 검증 | 웹 백테스팅 | 과거 적중률/수익률 | 시그널 신뢰도 판단 |

## 이탈 방지 포인트

| 이탈 시점 | 원인 | 대응 |
|-----------|------|------|
| 알림 무시 | 시그널이 매번 비슷하거나 적중률 낮음 | 스코어 임계값 조정, 고품질 시그널만 알림 |
| 대시보드 미접속 | 로딩 느림, 정보 과다 | < 3초 로딩, 핵심 정보 우선 표시 |
| 백테스팅 불신 | 과적합 의심 | Walk-forward 검증, 통계적 유의성 표시 |
| 시스템 방치 | 투자 관심 감소 | 주간 요약 리포트 (시그널 성과 추적) |
