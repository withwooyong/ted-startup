# PRD — 공매도 커버링 시그널 탐지 시스템

## 1. 제품 비전

> 한국 주식시장의 대차잔고 데이터를 자동 분석하여 숏커버링 진입 시점을 포착하는 개인 투자 참고 도구.

**한 줄 요약**: "오늘 대차잔고가 급감한 종목, 텔레그램으로 받아보세요."

## 2. 목표 사용자

| 구분 | 설명 |
|------|------|
| 유형 | 개인 투자자 (본인, 1인) |
| 투자 스타일 | 중단기 (5~20 거래일) |
| 기술 수준 | 개발자 (자체 운영 가능) |
| 분석 수준 | 기본 재무/기술적 분석 가능 |

## 3. 핵심 가치 제안

1. **자동화**: 매일 2시간 소요되던 수동 분석 → 배치 자동화 0분
2. **시그널**: 단순 데이터 나열이 아닌, 의미 있는 패턴 탐지
3. **검증**: 백테스팅으로 시그널 신뢰도 객관적 검증
4. **알림**: 텔레그램으로 실시간 전달

## 5. KPI

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 시그널 적중률 | ≥ 60% (5거래일 양의 수익률) | 백테스팅 리포트 |
| 배치 정시 실행률 | ≥ 95% | Spring Batch 메타데이터 |
| 대시보드 로딩 시간 | < 3초 | 브라우저 Performance API |
| 텔레그램 알림 도달률 | 100% | Telegram Bot API 응답 |
| 데이터 수집 커버리지 | KOSPI+KOSDAQ 전 종목 | 수집 건수 vs KRX 상장 종목 수 |
