# 대차거래 데이터 기반 공매도 커버링 시그널 탐지 시스템

한국 주식시장(KOSPI/KOSDAQ)에서 대차잔고 급감, 추세전환, 숏스퀴즈 패턴을 분석하여
상승 가능성이 높은 종목을 탐지한다.

## 도메인 제약
- 개인 투자자(본인)의 참고용 내부 도구
- "투자 자문" 아님, "정보 제공 도구"임
- 모든 시그널에 면책 고지 필수
- KRX 공개 데이터만 사용 (v1)

## 데이터 소스
- KRX 공매도 통계 (일별 공매도 거래대금, 대차잔고)
- KRX 시세 정보 (종가, 거래량, 시가총액)

## 핵심 기능 (v1 MVP)
1. 일 1회 배치로 KRX 데이터 수집
2. 시그널 탐지 (대차잔고 급감, 추세 전환, 숏스퀴즈 스코어링)
3. 백테스팅 (과거 3년)
4. 대시보드 + 텔레그램 알림

## 기술 스택
- Backend: Spring Boot 3.4 + Java 21
- DB: PostgreSQL 16 (시계열 파티셔닝)
- Frontend: Next.js 15 + Recharts
- 배치: Spring Batch
