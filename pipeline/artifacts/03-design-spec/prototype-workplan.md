# MVP UI 프로토타입 작업계획서

> 목적: shadcn/ui + TDS 디자인 토큰 적용된 최종 UI를 HTML로 미리 확인
> 생성일: 2026-04-16
> 산출물: `prototype/index.html` (단일 파일, 브라우저에서 바로 열기)

---

## 1. 목표

Design 단계 산출물(기능명세 + 디자인토큰 + 컴포넌트 명세)을 기반으로
**실제 구현 결과와 동일한 룩앤필의 인터랙티브 HTML 프로토타입**을 생성한다.

- 백엔드 없이 샘플 데이터로 동작
- shadcn/ui 스타일 (Tailwind CSS) 적용
- 다크모드 토글
- 반응형 (모바일/데스크톱)
- 3개 페이지 탭 전환

## 2. 기술 스택 (프로토타입 전용)

| 항목 | 선택 | 비고 |
|------|------|------|
| HTML | 단일 파일 | CDN 로드 |
| CSS | Tailwind CSS 3 (CDN) | shadcn/ui 스타일 재현 |
| 차트 | Chart.js (CDN) | Recharts 대신 (React 불필요) |
| 폰트 | Pretendard (CDN) | TDS 기본 폰트 |
| 아이콘 | Lucide Icons (CDN) | shadcn/ui 기본 아이콘 |
| JS | Vanilla JavaScript | 프레임워크 없음 |

## 3. 구현 범위

### 3.1 P1: 대시보드 (메인)

| # | 항목 | 상세 |
|---|------|------|
| 1 | 글로벌 네비게이션 | 로고 + 메뉴 (대시보드/백테스팅) + 다크모드 토글 |

| 3 | 시그널 필터 탭 | 전체/급감/추세전환/숏스퀴즈 + 건수 배지 |
| 4 | 시그널 리스트 (8~10개) | 종목명, 코드, 시그널타입, 스코어, 등급배지, 변동률 |

## 5. 인터랙션

| # | 동작 | 구현 방법 |
|---|------|-----------|
| 1 | 페이지 전환 | JS 탭 전환 (P1↔P2↔P3) |
| 2 | 시그널 필터 | JS로 리스트 필터링 |
| 3 | 종목 카드 클릭 | P2로 전환 + 해당 종목 데이터 표시 |
| 4 | 차트 기간 변경 | Chart.js 데이터 업데이트 |
| 5 | 다크모드 토글 | `document.documentElement.classList.toggle('dark')` |
| 6 | 정렬 변경 | JS로 카드 순서 재배치 |
| 7 | 뒤로 가기 | P2 → P1 전환 |
