| # | 페이지 | URL | 대응 US | 설명 |

```
┌──────────────────────────────────────────────────────┐
│  🔍 공매도 커버링 시그널        [날짜 선택] [새로고침] │
├──────────────────────────────────────────────────────┤
│  ┌─────────┐ ┌─────────┐ ┌─────────┐                │
│  │ 전체 12 │ │ 급감  5 │ │ 추세  4 │ │ 숏스퀴즈 3 │ │
│  └─────────┘ └─────────┘ └─────────┘                │
├──────────────────────────────────────────────────────┤
│  시그널 리스트                        [스코어순 ▼]   │
│  ┌──────────────────────────────────────────────┐    │
│  │ 🚀 A  카카오 (035720)          스코어 91     │    │
│  │      대차잔고 -18.5% | 거래량 +250%         │    │
│  ├──────────────────────────────────────────────┤    │
│  │ 🔴    삼성전자 (005930)        스코어 85     │    │
│  │      대차잔고 -15.3% | 3일 연속 감소         │    │
│  ├──────────────────────────────────────────────┤    │
│  │ ...                                          │    │
│  └──────────────────────────────────────────────┘    │

```

### 기능 상세

| 요소 | 동작 | 비고 |
|------|------|------|
| 날짜 선택 | 과거 날짜 시그널 조회 | 기본값: 오늘 |
| 시그널 탭 필터 | 전체/급감/추세전환/숏스퀴즈 | 탭 전환 시 리스트 필터 |
| 정렬 | 스코어순(기본)/감소율순/종목명순 | 드롭다운 |
| 시그널 카드 클릭 | 종목 상세 페이지(P2)로 이동 | |
| 등급 배지 | A(빨강)/B(주황)/C(노랑)/D(회색) | 숏스퀴즈 스코어 기반 |

### 빈 상태 (Empty State)
- 시그널 0건: "오늘은 탐지된 시그널이 없어요. 내일 다시 확인해 보세요."
- 데이터 미수집: "아직 데이터가 수집되지 않았어요. 배치 실행 상태를 확인해 주세요."

### 에러 상태
- API 실패: "데이터를 불러오지 못했어요. 잠시 후 다시 시도해 주세요." + 재시도 버튼
- 네트워크 오류: ErrorBoundary 폴백 UI

### 기능 상세

| 요소 | 동작 |
|------|------|
| 기간 선택 탭 | 차트 조회 기간 변경 (1/3/6/12개월) |

| 스코어 근거 | 각 지표별 점수 breakdown |

### 빈 상태
- 종목 없음 (잘못된 code): "해당 종목을 찾을 수 없어요." + 대시보드 이동 버튼
- 데이터 부족 (상장 직후): "분석에 필요한 데이터가 아직 부족해요. (최소 20거래일 필요)"

## 5. 공통 컴포넌트

### 5.1 글로벌 네비게이션

```
┌─────────────────────────────────────────┐
│  📊 시그널 탐지기   [대시보드] [백테스팅] │
│                     [시장동향] [설정]     │
└─────────────────────────────────────────┘
```

### 5.2 면책 고지 (Footer)

모든 페이지 하단에 고정:
> ⚠️ 본 시스템은 투자 자문이 아닌 정보 제공 목적의 참고 도구입니다.
> 모든 투자 판단과 그에 따른 결과의 책임은 투자자 본인에게 있습니다.
> 과거 성과가 미래 수익을 보장하지 않습니다.

### 5.3 로딩 상태

- 스켈레톤 UI (Suspense fallback)
- 차트: 로딩 스피너 + "차트를 불러오는 중이에요"

### 5.4 에러 바운더리

- ErrorBoundary 래핑 (페이지 단위)
- 폴백: "문제가 발생했어요. 새로고침하거나 잠시 후 다시 시도해 주세요."

## 6. 접근성 (WCAG 2.1 AA)

| 항목 | 적용 |
|------|------|
| 색상 대비 | 4.5:1 이상 (텍스트), 3:1 이상 (대형 텍스트) |
| 키보드 탐색 | 모든 인터랙티브 요소 Tab 접근 가능 |
| 스크린 리더 | 차트에 aria-label, 테이블에 caption |
| 포커스 표시 | focus-visible 스타일 적용 |
| 대체 텍스트 | 차트 하단에 데이터 테이블 토글 제공 |
